# 生成题答案位置设计

## 目标

修复本地规则生成题中单选题正确答案总是 `A` 的问题。生成的单选题需要展示 A-D 选项，并让正确答案位置根据题目内容变化，避免用户形成“默认选 A”的错误预期。

## 范围

- 优先修复知识内容生成题的单选题。
- 题目标题中包含 A-D 选项文本，保证用户作答时能看到选项。
- 正确答案字母根据内容确定，结果可复现，但不会固定为 A。
- 不改变人工导入 Markdown 中用户已经指定的答案。

## 策略

使用确定性“洗牌”：根据题目标题和内容生成偏移量，把正确选项插入 A-D 的不同位置，其余干扰项保持完整。这样同一份学习资料反复生成结果稳定，便于测试和复核；不同资料会得到不同正确选项位置。

## 验证

- 生成多份 HashMap 单选题，正确答案集合不能只有 `A`。
- 生成的单选题题干包含 A-D 选项。
- 正确答案字母对应的选项文本必须包含真实答案 `0.75`。
